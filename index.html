Key concepts

  • Semantic elements: <header>, <nav>, <main>, <article>, <section>, <aside>, <footer>.
  • Accessibility: landmarks, skip links, ARIA labels where necessary, meaningful alt text, and proper heading order.
  • SEO basics: descriptive <title>, <meta description>, heading hierarchy, and crawlable content.

Activity

Classroom Activity

Follow the repository instructions: create index.html that demonstrates a fully semantic HTML5 structure, uses accessible landmarks, contains clear headings in the correct order, and includes descriptive text for both users and search engines.

Requirements
  1. Use semantic HTML5 only — no CSS or JavaScript at this stage.
  2. Provide a clear page title and meta description.
  3. Implement a skip link and make main sections discoverable with headings.
  4. Include at least one <article> and one <aside> for related content.
  5. Submit index.html to the GitHub Classroom assignment repository before the deadline.

Deliverables

What to include in your submission:

  • index.html — the single HTML file with semantic structure and accessibility features.
  • A short README note in the PR describing your structure decisions (headings, landmarks, and accessibility choices).

Tips

  • Start with a clear <title> and <meta name="description">.
  • Use a single <h1> per page describing the main topic.
  • Ensure headings follow a logical, nested order.
